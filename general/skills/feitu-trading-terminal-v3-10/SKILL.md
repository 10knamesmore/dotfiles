# 非凸智能交易终端 V3.10 券商版 — 使用手册（文字版）

这是非凸智能交易终端 V3.10 **券商版**客户端的官方使用手册，**所有 73 张界面截图已用详细文字转述**，
非多模态模型也能完整理解每个界面的控件位置与操作步骤。

> **券商版与完整版的区别**：券商版去掉了期货账户、期货交易、期货拆单交易、移仓交易、
> 绩效分析等模块，只保留证券（普通账户）+ 信用账户两条线，以及智能交易、增强交易、
> 证券增强文件单等共用功能。如果用户在客户端里看不到上述被去掉的模块，属于券商版正常。

## 图片转述格式约定

文档中的每张原始截图位置都放置一个 blockquote 块（`> ...`），例如：

```
> **界面 4：登录界面**
> 「系统登录」窗口的整体截图。布局：
> - 1.【记住密码】复选框（账号密码下方左侧）
> - 2.【自动登录】复选框（记住密码右侧）
> - 3.【下拉】按钮（账号输入框右侧）
> - 6.【连接设置】按钮（登录按钮右下方蓝色文字）
> ...
> 与正文配合：正文"点击 3.【下拉】"指账号框右侧那个按钮。
```

- blockquote 里 `1./2./3./...` 的编号与**正文中的引用编号一一对应**——
  正文写"点击 3.【下拉】"时，去 blockquote 找标号为 `3.` 的条目即可定位按钮。
- 本 skill 目录里**不存图片文件**，blockquote 末尾**没有** `![imageN.png]` 链接；
  全部用文字描述，非多模态模型直接阅读即可。

## 使用建议

1. 用户问题里出现"非凸""智能交易终端""客户端""母单""拆单""底仓增强""券商版"等关键词时，
   先查上面的章节路由表，命中哪一行就 Read 那个 reference 文件。
2. 涉及具体按钮位置（"3.【下拉】是哪个"），找到对应 reference 后**在 blockquote 中搜编号**即可。
3. 多个章节可能都涉及"母单上传"——优先按 heading 路径精确匹配
   （拆单交易 vs 底仓增强交易 的母单上传是不同界面）。
4. 操作流程类问题（"母单上传要走哪几步"），阅读对应章节的正文 + blockquote，按顺序组织步骤回答。
5. 用户如果问"期货怎么交易""绩效分析在哪""移仓怎么用"，请告知：**本手册是券商版，不含这些功能**。
   如果他们确实需要，应使用完整版客户端或查阅 V3.10 完整版手册。
6. blockquote 中的界面描述精确到位置、颜色、按钮文字、示例值，但若某个细节用户存疑，
   建议对照原 docx `非凸智能交易终端使用手册_V3.10_券商版.docx` 中对应截图核实。
