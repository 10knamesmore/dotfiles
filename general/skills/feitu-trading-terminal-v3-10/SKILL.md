# 非凸智能交易终端 V3.10 — 使用手册（纯文字版）

这是非凸智能交易终端 V3.10 券商版客户端的官方使用手册，**所有界面截图已用详细文字转述**，
不依赖任何图片文件，任何模型（含非多模态）都能完整理解每个界面的控件位置与操作步骤。

## 图片转述格式约定

文档中的每张原始截图位置都放置一个 blockquote 块（`> ...`），例如：

```
> **图 4：登录界面**
> 这是登录窗口的截图。布局：
> - 顶部：账号 / 密码输入框
> - 1.【记住密码】复选框（账号密码下方左侧）
> - 3.【下拉】按钮（账号输入框右侧）
> - 6.【连接设置】按钮（左下角）
> ...
> 与正文配合：正文"点击 3.【下拉】"指账号框右侧那个按钮。
```

- blockquote 里 `1./2./3./...` 的编号与**正文中的引用编号一一对应**——
  正文写"点击 3.【下拉】"时，去 blockquote 找标号为 `3.` 的条目即可定位按钮。
- 本技能为**纯文字说明书**，blockquote 内**不含图片链接**，所有视觉信息已转述为文字。

## 使用建议

1. 用户问题里出现"非凸""智能交易终端""客户端""母单""拆单""底仓增强""移仓"等关键词时，
   先查上面的章节路由表，命中哪一行就 Read 那个 reference 文件。
2. 涉及具体按钮位置（"3.【下拉】是哪个"），找到对应 reference 后**在 blockquote 中搜编号**即可。
3. 多个章节可能都涉及"母单上传"——优先按 heading 路径精确匹配
   （拆单 vs 底仓增强 vs 移仓 的母单上传是三个不同界面）。
4. 操作流程类问题（"母单上传要走哪几步"），阅读对应章节的正文 + blockquote，按顺序组织步骤回答。
5. 若用户问到 V3.10 与 V3.11 的差异，优先回答当前手册（V3.10）内容，并提示用户可参考 `feitu-trading-terminal` 技能查 V3.11 细节。
