## 拆单交易

【拆单交易】提供拆单算法交易服务，支持单票手动委托和文件批量委托，支持普通交易和信用交易。

### 界面介绍

> **界面 38：拆单交易主界面分区**
> 「智能交易 > 拆单交易」主界面（整体，使用红色矩形框标注五大分区）。布局：
> - 左上 1.【下单窗口】：红框区域，顶部有「文件上传 / 母单上传」切换 tab，下方列出「交易账户」「篮子名称」「证券代码」等表单字段以及蓝色「一键提交」按钮。
> - 顶部中部 2.【母单列表】：红框区域，列头依次为「交易帐户 / 母单编号 / 证券代码 / 证券名称 / 交易所 / 委托时间 / 委托数量 / 算法类型 / 买卖标识 / 开始时间 / 结束时间 / 成交数量 / 完成程度 / 母单状态 / 所属篮子 / 延长结束时间」，行内有 FT_AI_WAP_PLUS 算法、买入/卖出标签等。
> - 中部下方 3.【子单列表】：红框区域，列头「交易帐户 / 单单编号 / 证券代码 / 证券名称 / 交易所 / 委托时间 / 报单状态 / 委托数量 / 报单标识 / 委托价格 / 客单编号 / 成交数量 / 成交均价」，可显示「全部成交」「废单」等状态。
> - 右侧 4.【监控中心】：红框区域，顶部「拆单监控中心」标题，下方有「总交易额」「撤单率」「总母单数」「交易账户数」「进度」「买交易额」「卖交易额」「错废单率」等指标，含「账户监控/篮子监控」tab。
> - 右下角 5.【账户异常提醒】：红框区域，提示「普通测试 经纪测试 等 N 个账户尽请管理」黄色文字 + 蓝色【查看】链接（用于快速跳转账户管理）。
> 与正文配合：对应正文「1.【下单窗口】2.【母单列表】3.【子单列表】4.【监控中心】5.【账户异常提醒】」五大分区编号。

1.【下单窗口】支持文件和母单上传，详细介绍请移至下文的7.1.2母单上传和7.1.3文件上传。
2.【母单列表】显示母单回报内容，包括母单委托状态，成交数量，完成度等，数据实时刷新，可以左右滑动查阅更多信息。支持启动，暂停和撤销操作，支持数据导出操作。
3.【子单列表】单击选中目标母单后，在此区域显示对应子单详情，包括子单委托状态，委托价格，股数和成交数量等，可以左右滑动查阅更多信息。
4.【监控中心】显示所有交易账户的拆单算法交易进度整体情况，包含账户和篮子级别监控。
指标说明：
总交易额 ：当日总计已成交金额
撤单率 ：子单撤单数占总子单数的比率
总母单数 ：当日输入的母单总数量
交易账户数 ：当日用户有母单输入的交易账户数量
进度 ：所有拆单母单的执行总进度
买交易额 ：买入委托的成交总额
卖交易额 ： 卖出委托的成交总额
错废单率 ：子单错废单数占总子单数的比率
买入计划 ：买入已成交额 + 未成交部分 × 昨收价
买入比例 ：买交易额 / 买入计划
卖出计划 ：卖出已成交额 + 未成交部分 × 昨收价
卖出比例 ：卖交易额 / 卖出计划
总比例 ：(买交易额 + 卖交易额) / (买入计划 + 卖出计划)
多空暴露 ：(买交易额 - 卖交易额) / (买交易额 + 卖交易额)
5.【账户异常提醒】显示当前已登陆但登录失败的交易账户报错信息，支持点击【查看】快速跳转到【账户管理】界面查看和修改账户登录设置。

字段释义：
A列 ：证券代码，必填，建议保留交易代码后缀。交易所代码说明，SH: 上交所，SZ: 深交所，BSE: 北交所，CFFEX: 中金所，SHFE: 上期所，DCE: 大商所，CZCE: 郑商所，SGE: 上海黄金交易所，INE: 上海国际能源中心
B列 ：策略类型，必填，支持输入数字或字符串。100：FT_AI_WAP_PLUS，300：FT_AI_WAP，1000: TWAP，1001: VWAP，1002: POV
注意：策略版本更新后，请联系运维专员确认策略ID和名称。TWAP/VWAP/POV属于被动算法类型，被动算法的绩效预期显著低于智能算法，请您谨慎使用!
C列 ：买卖方向，必填。普通买入/担保品买入输入“buy”或1，普通卖出/担保品卖出输入“sell”或2，融资买入输入“mb”或11，融券卖出输入“ss”或12，买券还券输入"rb”或21，卖券还款输入“rs”或22
D列 ：委托数量，必填
E列 ：开始时间，必填
F列 ：结束时间，必填
G列 ：资金账号，必填
H列 ：篮子名称，可以为空
I列 ：客户母单编号，可以为空
J列 ：其他参数，可以为空
注意：起止时间必须在交易时间段内，9: 30~14: 57。时间格式是整数类型，例如:“93500"表示9:35:00。

> **界面 47：目标持仓 CSV 字段示例**
> 「position_csv_to_do 文件夹」目标持仓 CSV 模板（整体，Excel 表格）。布局：
> - 列头（A~E）：无具体表头文字，仅展示数据行。
> - 第 1 行示例：A=5 / B=20000 / C=130000 / D=150000 / E=800167。
> - 第 2 行示例：A=600001 / B=200 / C=130000 / D=150000 / E=800167。
> - 第 3 行（末尾标志行）：五列均填 0——A=0 / B=0 / C=0 / D=0 / E=0。
> - 数字右对齐，无 F 列（F 列空）。
> - 字段含义：A=证券代码（无后缀）、B=目标持仓数量、C=开始时间（130000=13:00:00）、D=结束时间、E=资金账号。

字段释义：
A列 ：证券代码，必填，无须交易所后缀
B列 ：目标持仓数量，必填
C列 ：开始时间，必填
D列 ：结束时间，必填
E列 ：资金账号，必填
F列 ：篮子名称，可以为空
注意：起止时间必须在交易时间段内，9: 30~14: 57。时间格式是整数类型，例如:“93500"表示9:35:00。最后一行需为五列均需填 0.
