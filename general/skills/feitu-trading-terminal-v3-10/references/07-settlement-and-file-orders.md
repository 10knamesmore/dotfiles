# 拆单结算 / 底仓增强结算 / 增强交易 / 证券增强文件单

## 拆单结算

在【拆单结算】页面，可以查询不同交易账户在指定日期下的拆单算法交易结算统计数据。

> **界面 64：拆单结算主界面**
> 「账户管理 > 拆单结算」页面（顶部标签页区域可见「账户管理 / 模拟 / 拆单结算」等 tab，当前选中「拆单结算」）。布局：
> - 顶部条 1.【查询】：横向工具栏，左侧为「交易账户」下拉框（标签「模拟」）+ 蓝色「连接成功」状态标签；右侧为「时间范围」起止日期选择器（示例值 2024-11-01 至 2024-11-30）+ 蓝色「查询」按钮。整个查询条用红色虚线框框出，右侧标红色编号「1.【查询】」。
> - 左侧列 2.【绩效总览】：竖向汇总卡片，标题「数据统计」。指标按两列排列——左列「成交额 561.14 万」「总盈亏 1.12 万」（红色）「买入盈亏 1.12 万」（红色）；右列「母单数 3」「胜率 33.33%」「卖出盈亏 0」，盈亏数字红色高亮。
> - 中部上方 3.【日度指标详情】：横向表格，列头从左到右依次为「日期」「总盈亏」「买入金额」「买入盈亏」「卖出金额」「卖出盈亏」「BP」等。示例数据行 20241122 显示总盈亏 11151.81、买入金额 5611446.00、买入盈亏 11151.81、卖出金额/盈亏 0.00、BP 19.87，红色数字突出显示盈亏值。
> - 底部 4.【母单指标详情】：横向表格，顶部带筛选条「母单 ID / 证券代码」搜索框。列头依次为：日期、母单 ID、证券代码、买卖标识、报单数量、开始时间、结束时间、成交数量、成交均价、结算标识、结算基准价、盈亏、折算笔手。示例数据三行（20241122 三笔），买卖标识列以红色「买入」标签显示，结算标识列为绿色「FBOS」标签，盈亏列红绿数字混合（如 -222.03、11789.14、-418.29）。
> - 整体红色虚线框 + 红色编号 1/2/3/4 分布于四个区域，背景白色。
> 与正文配合：对应正文「1.【查询】」「2.【绩效总览】」「3.【日度指标详情】」「4.【母单绩效详情】」四个分区的说明。

1.【查询】，选择需要查询的交易账户和开始/结束日期。
2.【绩效总览】中的数据统计显示的是指定日期范围内对应交易账户拆单算法交易的结算绩效统计结果。
3.【日度指标详情】，显示在查询日期范围内对应交易账户的每日绩效统计结果。
4.【母单绩效详情】，显示查询日期范围内对应交易账户所有母单的明细情况，支持筛选。


## 底仓增强结算

在【底仓增强结算】页面，可以查询不同交易账户在指定日期下的底仓增强算法交易结算统计数据。

> **界面 65：底仓增强结算主界面**
> 「账户管理 > 底仓增强结算」页面，整体布局与拆单结算同构，仅指标项不同。布局：
> - 顶部条 1.【查询】：横向工具栏，左侧「交易账户」下拉框 + 蓝色「连接成功」标签；右侧「时间范围」起止日期选择器（示例 2024-11-01 至 2024-12-06）+ 蓝色「查询」按钮。红色编号「1.【查询】」标注在右侧。
> - 左侧列 2.【绩效总览】：竖向汇总卡片，标题「数据统计」。指标两列排布——左列「成交额 38.58 万」「总盈亏 -189.58 元」（绿色，负值表示亏损）「买入盈亏 19.29 万」（红色）；右列「母单数 7」「胜率 14.29%」「卖出盈亏 19.29 万」（红色）。盈利红色 / 亏损绿色。
> - 中部上方 3.【日度指标详情】：横向表格，列头从左到右依次为「日期」「计划金额」「买入金额」「卖出金额」「收益」「成交即收益率」「底仓收益率」「多头敞口金额」「空头敞口金额」「交易敞口金额」。示例三行 20241122 / 20241203 / 20241206，收益列均显示绿色负值（-121.41、-12.14、-56.03），成交即收益率/底仓收益率以 bp 单位显示。
> - 底部 4.【母单指标详情】：横向表格，顶部带「母单 ID / 证券代码」搜索框。列头包括：日期、母单 ID、证券代码、证券名称、计划数量、买入数量、买入金额、卖出数量、卖出金额、母单盈亏、成交即收益率、底仓收益率、敞口字段。示例 6 行明细，证券名称列含「白马股份 / 浦发银行 / 平安银行」，母单盈亏列均为绿色负值（亏损），敞口列右侧带浅红/浅绿背景色块凸显正负。
> - 整体仍以红色虚线框分区，标注 1/2/3/4 红色编号。
> 与正文配合：对应正文「1，2，3 和 4 与【拆单结算】页面一致，日度和母单指标有所区别」一句——四个分区位置不变，但 3、4 区列头由拆单结算的盈亏类指标替换为底仓增强专属的「计划金额 / 收益率 / 敞口金额」类指标。

1，2，3和4与【拆单结算】页面一致，日度和母单指标有所区别。


## 增强交易

【增强交易】支持在客户端内运行用户编写的交易策略，提供交易所需要的完备接口，包括下单撤单、查询等能力。

> **界面 66：增强交易主界面**
> 「账户管理 > 增强交易」页面（顶部 tab 区域可见「账户管理 / 增强交易」等标签，当前选中「增强交易」）。整个界面被红色虚线框分为四个编号区域。布局：
> - 左侧列 1.【策略列表】：竖向窄面板，顶部「策略列表」标题 + 「刷新」文字按钮，下方以网格形式展示策略图标。当前列出一项「demo」（文档样式图标 + 策略名 demo + 长 sha256 字符串如 `9442b2c0be84e19c3c8704c100780f01764`），双击图标即可启动该策略。红色编号「1.【策略列表】」标注在面板下方。
> - 顶部主区域 3.【运行中策略】：横向表格，顶部「运行中策略」标题，列头从左到右依次为「策略名称」「策略文件名」「运行状态」「开始时间」「结束时间」「操作」。当前显示「暂无数据」占位文本。红色编号「3.【运行中策略】」标注在右侧。
> - 中部弹窗 2.【新建策略】：白底浮动对话框，标题「新建策略交易」+ 右上角关闭「×」。表单字段含「策略名称」（示例值 demo）、「sha256」（只读哈希），参数表格三列「参数名称 / 值 / 描述」，示例两行参数「config_dir = a/b/c（配置文件目录）」「config2」。底部「取消」灰色按钮 + 蓝色「确认」主按钮。外圈有红色矩形高亮框，红色编号「2.【新建策略】」标注在弹窗下方。
> - 底部全宽区 4.【数据查询】：横向表格，顶部带 tab 切换「篮单列表 / 篮单日志」（当前选中「篮单列表」），下方有筛选条「篮单ID/证券代码」搜索框 + 多个状态筛选下拉框（「全部交易账户」「全部报单状态」），右侧 tab 切换「跑动中」（蓝色高亮）/「撤单中」「撤单」「成交」「废单」「未发」。表格列头横向铺满：子单编号、交易账户ID、策略名称、证券代码、证券名称、交易账户名称、委托编号、买卖标识、委托价格、委托数量、成交均价、成交数量、成交状态、委托时间、市场、废单原因。当前显示「暂无数据」。红色编号「4.【数据查询】」标注在区域下方。
> 与正文配合：对应正文「在 1.【策略列表】双击策略图标，弹出 2.【新建策略】窗口」「3.【运行中策略】显示在运行的策略」「4.【数据查询】支持查询委托记录和策略日志」三段描述——弹窗 2 即由双击区域 1 中的策略图标触发。

在1.【策略列表】双击策略图标，弹出2.【新建策略】窗口，支持再次编辑策略参数。
3.【运行中策略】显示在运行的策略，支持暂停和撤销操作。
4.【数据查询】支持查询委托记录和策略日志。
注意：策略需要向运维专员报备后才可以使用。


## 证券增强文件单

【证券增强文件单】提供以文件单形式对接外部交易系统的交易执行管理服务。

> **界面 67：证券增强文件单主界面**
> 「账户管理 > 证券增强文件单」页面（顶部 tab 区域可见「账户管理 / 拆单交易 / 底仓增强交易 / 证券增强文件单」标签，当前选中「证券增强文件单」）。整个界面被红色虚线框分为三个编号区域。布局：
> - 顶部条 1.【运行设置】：横向工具栏，左侧为蓝色「运行中」状态徽标 + 绿色「停止」按钮（带方形 icon），中部为路径输入框（示例 `C:\非凸智能交易终端(fts)\增强文件单`）+ 文件夹选择 icon + 蓝色「下载模版」按钮；右侧依次为「扫描频率」输入框（默认 1000 毫秒）、「资产导出频率？」输入框（默认 10000 毫秒，带问号 tooltip），最右侧勾选框「登录自动启动」（已勾选蓝色）。外圈有红色矩形高亮框，红色编号「1.【运行设置】」标注在右上方。
> - 中部主区域 2.【数据统计】：上半部分为横向指标条，10 项统计依次为「母单委托数 0」「母单失败数 0」「母单确认数 0」「母单移除数 0」「交易账户数 0」「子单委托数 0」「子单错废单数 0」「子单撤单数 0」「子单错废单率 0.00%」「子单撤单率 0.00%」，数值为大字号黑色。下半部分为子 tab 切换「待确认母单」（蓝色高亮当前选中）/「已确认母单」/「已移除母单」+ 蓝色勾选框「自动确认」（已勾选）+ 两个白底按钮「确认」「移除」。下方表格列头：勾选列、自定义委托编号、账户名称、证券代码、交易所、买卖标识、委托数量、开始时间、结束时间、篮子名称、算法类型、算法参数。当前显示「暂无数据」。红色编号「2.【数据统计】」标注在右上方。
> - 底部 3.【运行日志】：左上「运行日志」标题 + 灰字「显示最近 100 条」。下方控制条「日志目录」标签 + 路径输入框（`C:\非凸智能交易终端(fts)\log`）+ 文件夹 icon + 问号 icon + 「内容」搜索框 + 放大镜 icon + 「全部类型」下拉框。日志表格列头「时间 / 类型 / 内容」三列，两条示例日志：「16:21:01  INFO（蓝色徽标）  证券增强文件单启动」「16:20:56  INFO  证券增强文件单将于 5 秒后自动启动」。红色编号「3.【运行日志】」标注在右上方。
> 与正文配合：对应正文「1.【运行设置】支持设置增强文件单路径、文件扫描和导出频率」「2.【数据统计】显示增强文件单的交易统计数据」「3.【运行日志】支持查看增强文件单的运行日志信息」三段说明——下载模板按钮即位于区域 1 内。

1.【运行设置】支持设置增强文件单路径，文件扫描和导出频率。文件格式要求请点击【下载模板】查看。
2.【数据统计】显示增强文件单的交易统计数据。
3.【运行日志】支持查看增强文件单的运行日志信息。
注意：除证券增强文件单外，客户端还支持API交易，如有使用需求请联系运维专员。
